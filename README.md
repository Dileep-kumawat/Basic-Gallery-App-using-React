## ğŸ”§ Installation

Follow these steps to run this project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Dileep-kumawat/Basic-Gallery-App-using-React.git

2. **Go to project directory**

   ```bash
   cd Basic-Gallery-App-using-React
   ```
3. **Install dependencies**

   ```bash
   npm install
   ```
4. **Start development server**

   ```bash
   npm run dev
   ```

Now open your browser at `http://localhost:5173` (or the port Vite shows).
